      Listas de Definicion

      Terminos de Internte

      Interntet
      Red informática de nivel mundial que utiliza la línea telefónica para transmitir la información
      Navegador
      Programa que permite navegar por internet u otra red informática de comunicaciones
      HTML
      Lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto
      Wifi
      Tecnología que permite conectar diferentes equipos informáticos a través de una red inalámbrica de banda ancha
